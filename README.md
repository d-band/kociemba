Kociemba Algorithm
==================


## Install

```bash
npm install kociemba --save
```

## Usage

```javascript
const kociemba = require('kociemba');

kociemba.solve('DDBUUFBDDLRUBRDUBRLFFBFDUFBFLLFDLRRFRLDULRUULRLBRBBDUF');

// => "F2 U2 D' B D2 B2 L' D' F U2 F L2 B2 R2 L2 D R2 L2 F2 U2 L2 "
```
